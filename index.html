About Me

Through this internship, I learned many programming techniques, communication skills, and development methods used in a business environment. Some of the useful skills I learned at BJM include: git version control, static web design (HTML & CSS), dynamic web design (PHP, SQL Databases, & CMS), Search Engine Optimization, graphic design, photo and video production, and more. This internship had a huge impact on my career path and expertise in this field

I've also worked on some school projects covering the topics of mobile applications, database systems, and web development.
